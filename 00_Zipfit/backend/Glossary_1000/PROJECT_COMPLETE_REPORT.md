## 🚀 LLM 활용 가이드

### 1. RAG (Retrieval-Augmented Generation)
```python
# 사용자 질문 예시
query = "대항력이 뭔가요?"

# 벡터 검색
results = vector_db.similarity_search(query, k=3)

# 컨텍스트 구성
context = "\n".join([
    f"**{r['term']}**: {r['definition_easy']}"
    for r in results
])

# LLM 프롬프트
prompt = f"""
당신은 공공주택 전문 상담사입니다.
다음 용어 정보를 참고하여 답변하세요:

{context}

사용자 질문: {query}

답변 원칙:
1. 쉬운 말로 설명
2. 구체적 예시 포함
3. 관련 용어도 안내
"""
```

## 💡 활용 시나리오

### 시나리오 1: 챗봇 일반 질문
```
사용자: "전세랑 월세가 뭐가 다른가요?"

챗봇: 
전세는 큰 보증금을 맡기고 월세 없이 살다가 
나갈 때 보증금을 돌려받는 방식이에요.

월세는 적은 보증금 + 매달 집세를 내는 방식입니다.

예를 들어:
- 전세: 보증금 3억, 월세 0원
- 월세: 보증금 5천만원, 월세 60만원

관련 용어: 반전세, 임대차계약
```

### 시나리오 2: 법률 용어 설명
```
사용자: "대항력이 뭔가요? 중요한 건가요?"

챗봇:
대항력은 전입신고하고 집에 살면 
주인이 바뀌어도 계속 살 권리를 말해요.

중요한 이유:
1. 집이 경매로 넘어가도 계약 유지
2. 새 주인한테도 권리 주장 가능
3. 보증금 돌려받기 유리

꼭 필요한 조건:
✅ 전입신고 완료
✅ 실제 거주 중

관련: 확정일자, 우선변제권
```

### 시나리오 3: 청약 절차 안내
```
사용자: "LH 청약 어떻게 하나요?"

챗봇:
LH 청약 5단계:

1️⃣ 모집공고 확인 (LH 홈페이지)
2️⃣ 자격 확인 (소득, 무주택기간)
3️⃣ 온라인 청약 (LH청약플러스)
4️⃣ 당첨자 발표 (1주일 후)
5️⃣ 서류 제출 & 계약

필요 서류:
- 주민등록등본
- 소득금액증명원
- 가족관계증명서

더 궁금하신 용어 있으세요?
```
