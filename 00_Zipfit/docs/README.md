# ZIPFIT 프로젝트 문서

## 프로젝트 개요

**서비스명**: 집핏 (ZIPFIT)

**슬로건**: "나에게 딱 맞는 집, ZIPFIT"

**핵심 가치**: 복잡한 공공주택 공고를 LLM과 RAG 기술로 분석하여, 사용자 맞춤형 주거 정보를 제공하는 원스톱 AI 에이전트 서비스

## 프로젝트 진행 상황

### Phase 1: 프로토타입 (진행 중)
- 범위: 서울 LH 임대 공고만
- 기능: 챗봇 + 공고 목록 + 상세보기
- 목표: 2025년 11월 23일 (일)까지

### Phase 2: 확장 (예정)
- 범위: 전체 지역, SH/GH 추가, 임대+분양
- 기능: 공고 비교, 자격 진단, 대출 정보 연동

### Phase 3: 고도화 (예정)
- 범위: 계약 지원, 알림 기능, 성능 최적화
- 기능: 전자계약 연동, 개인화 추천, 커뮤니티

## 기술 스택

### Frontend
- React + TypeScript + Vite
- shadcn/ui + Radix UI
- React Router
- TailwindCSS

### Backend
- FastAPI (또는 Flask) - 검토 중
- PostgreSQL + pgvector (Vector DB)
- Docker

### AI/LLM
- OpenAI GPT-4o-mini
- Sentence-Transformers (임베딩)
- RAG (Retrieval-Augmented Generation)

### Data Source
- data.go.kr API (LH, GH 공고)
- 크롤링 데이터 (2024~현재)
- Glossary (1000개 부동산 용어)
