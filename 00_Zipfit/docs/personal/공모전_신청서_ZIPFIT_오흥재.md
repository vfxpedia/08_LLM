### 시나리오 개요

ZIPFIT(집핏)은 LH, SH, GH 등 공공주택 공고를 통합하여, LLM과 RAG 기술을 활용해 사용자 맞춤형 주거 정보를 제공하는 AI 에이전트 서비스입니다.

복잡하고 이해하기 어려운 공공주택 공고문을 분석하여, 사용자의 조건에 맞는 공고를 추천하고, 자격 요건을 확인하며, 신청부터 계약, 대출까지 원스톱으로 안내합니다.

**핵심 서비스 흐름**:
AI 상담 → 자격 진단 → 공고 비교/추천 → 신청 안내 → 계약 지원 → 대출 정보 제공

### 제안 배경

#### (배경)

공공주택(LH, SH, GH)은 청년, 신혼부부, 저소득층 등 주거 취약계층에게 매우 중요한 주거 안정 수단입니다. 그러나 현재 공공주택 공고는 다음과 같은 문제점이 있습니다:

1. **복잡한 공고 구조**: 각 기관(LH, SH, GH)마다 공고 형식과 용어가 달라, 통합적으로 비교하기 어렵습니다.

2. **전문 용어의 범람**: "선계약후검증", "묵시적갱신", "계약보증금" 등 일반인이 이해하기 어려운 전문 용어가 많습니다.

3. **산재된 정보**: 자격 요건, 신청 방법, 대출 정보 등이 각각 다른 문서와 웹사이트에 흩어져 있어 정보 수집에 많은 시간이 소요됩니다.

4. **자격 판단의 어려움**: 본인이 해당 공고에 신청할 자격이 되는지 판단하기 위해서는 복잡한 소득 기준, 거주 요건 등을 일일이 확인해야 합니다.

이러한 문제로 인해 많은 국민들이 자신에게 적합한 공공주택 기회를 놓치거나, 부적격 신청으로 시간을 낭비하는 경우가 발생하고 있습니다.

#### (필요성)

**1. LLM만이 비정형 문서를 이해하고 설명할 수 있습니다**

공공주택 공고문은 PDF 형태의 비정형 문서로, 기관마다 구조가 다릅니다. 기존의 키워드 검색이나 필터링으로는 사용자가 원하는 정확한 정보를 제공하기 어렵습니다. LLM은 문서의 맥락을 이해하고, 사용자의 질문에 맞춰 필요한 정보를 추출하여 쉽게 설명할 수 있습니다.

**2. 사용자 맞춤형 정보 제공이 필수적입니다**

현재 공공주택 플랫폼들은 공고를 단순히 나열할 뿐, "내가 신청할 수 있는지", "어떤 공고가 나에게 유리한지"를 판단해주지 못합니다. AI 에이전트는 사용자의 나이, 소득, 가족 구성 등을 고려하여 맞춤형 공고를 추천하고, 자격 요건을 자동으로 확인할 수 있습니다.

**3. 원스톱 서비스로 국민 편의성을 극대화합니다**

공고 검색부터 신청, 계약, 대출까지 여러 단계를 거쳐야 하는 번거로움을 AI 에이전트가 하나의 대화 창에서 모두 처리할 수 있도록 지원합니다. 이는 디지털 격차가 있는 고령층이나 정보 접근성이 낮은 사회 취약계층에게 특히 유용합니다.

**4. 사회적 가치 실현**

공공주택은 단순한 부동산 정보가 아니라, 국민의 주거권과 직결된 사회 안전망입니다. ZIPFIT은 모든 국민이 공평하게 공공주택 정보에 접근하고, 자신에게 맞는 주거 기회를 찾을 수 있도록 지원함으로써 주거 불평등 해소에 기여합니다.

### 기대효과

#### 1. 경제적 효과

**공고 검색 및 분석 시간 대폭 단축**
- 기존: 여러 사이트를 방문하여 공고를 일일이 확인 (평균 2~3시간 소요)
- ZIPFIT: AI 대화를 통해 맞춤형 공고 추천 (평균 10~15분 소요)
- 효과: 시간 비용 약 90% 절감

**부적격 신청 감소로 행정 비용 절감**
- 자격 요건 자동 확인으로 부적격 신청 사전 방지
- 공공기관의 심사 업무 부담 경감

**대출 정보 통합 제공으로 금융 접근성 향상**
- 대출 상품 비교 및 신청 절차 안내
- 주거 비용 부담 완화

#### 2. 사회적 가치

**주거 취약계층의 주거 안정 지원**
- 청년: 행복주택, 매입임대 등 청년 전용 공고 맞춤 추천
- 신혼부부: 신혼희망타운, 신혼부부 우선 공급 공고 안내
- 저소득층/사회 취약계층: 영구임대, 국민임대 등 소득 기준 맞춤 공고 제공
- 고령층: 복잡한 디지털 절차를 대화형 AI로 간소화

**공공주택 정보 접근성 향상**
- 디지털 격차 해소: 복잡한 웹사이트 탐색 없이 대화만으로 정보 획득
- 정보 비대칭 완화: 모든 국민이 동일한 수준의 정보 접근 가능

**복잡한 공고문의 이해도 향상**
- 전문 용어를 쉬운 언어로 설명
- 사용자 눈높이에 맞춘 맞춤형 해설

#### 3. 국민에게 미치는 영향

**신청 성공률 향상**
- 자격 요건 사전 확인으로 적격 공고에만 신청
- 맞춤형 추천으로 경쟁률 낮은 공고 발굴 가능

**주거 복지 체감도 증가**
- 공공주택이 "어렵고 복잡한 절차"에서 "쉽고 편리한 서비스"로 전환
- 공공주택 신청률 증가 → 주거 안정 기여

**원스톱 서비스로 편의성 극대화**
- 공고 검색 → 자격 확인 → 신청 안내 → 계약 지원 → 대출 정보 제공까지 하나의 플랫폼에서 완결
- 여러 사이트를 오가며 정보를 수집하는 번거로움 해소

**향후 확장 가능성**
- 연결형 확장: 국토부 전자계약 시스템 연동, 지자체 복지 서비스 연계
- 융합형 확장: 지도 API 연동(주변 인프라, 학군 정보), 커뮤니티 리뷰 통합

**세부 시나리오 설명**

위 두 시나리오는 ZIPFIT의 핵심 기능을 보여줍니다:

1. **AI 상담**: 자연스러운 대화로 사용자 조건 파악
2. **자격 진단**: 공고문 분석을 통한 자동 자격 확인
3. **공고 비교/추천**: 사용자 맞춤형 공고 추천 및 장단점 분석
4. **신청 안내**: 단계별 신청 절차 상세 안내
5. **대출 정보**: 정부 지원 대출 상품 연동 정보 제공
6. **외부 연결**: 실제 신청/상담 사이트로 원클릭 연결

모든 과정이 하나의 대화 창에서 원스톱으로 이루어지며, 사용자는 복잡한 웹사이트를 탐색하거나 여러 문서를 읽을 필요 없이 필요한 정보를 빠르게 얻을 수 있습니다.

### 향후 확장 계획

**연결형 확장**:
- 국토부 부동산 전자계약 시스템 연동 → 온라인 계약 체결 지원
- 지자체 복지 서비스 연계 → 주거급여, 생활지원금 등 통합 안내

**융합형 확장**:
- 지도 API 연동 → 주변 인프라(학교, 병원, 마트 등) 정보 제공
- 생활 인프라 분석 → 교통 접근성, 학군 정보, 생활 편의 점수 제공
- 커뮤니티 리뷰 통합 → 실거주자 후기 및 단지 평가 제공
