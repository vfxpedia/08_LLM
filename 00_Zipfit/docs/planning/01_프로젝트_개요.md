# 01. 프로젝트 개요

## 1.1 서비스 소개

### 서비스명
**집핏 (ZIPFIT)**

### 슬로건
**"나에게 딱 맞는 집, ZIPFIT"**

### 핵심 가치
어려운 공공주택 공고 내용과 복잡한 문서를 압축하여, 사용자가 원하는 정보를 정확하고 빠르게 전달하는 AI 에이전트 서비스

---

## 1.2 프로젝트 목적

ZIPFIT은 LH, SH, GH 등 공공주택 기관의 분산된 공고를 통합하고, LLM과 RAG 기술을 활용하여 사용자 맞춤형 주거 정보를 제공합니다.

**핵심 목표**:
1. 복잡한 공고문을 쉽게 이해할 수 있도록 AI가 설명
2. 사용자 조건에 맞는 공고를 자동으로 추천
3. 공고 검색부터 신청, 계약, 대출까지 원스톱 서비스 제공
4. 모든 국민이 공평하게 공공주택 정보에 접근할 수 있도록 지원

---

## 1.3 주요 기능

### 1.3.1 AI 상담 (Conversational Interface)

**기능**: 자연스러운 대화를 통해 사용자의 주거 니즈 파악

**주요 특징**:
- 사용자 친화적인 대화형 인터페이스
- 나이, 소득, 가족 구성 등 개인 정보 수집
- 맥락을 이해하는 자연어 처리

**예시**:
```
사용자: "서울에서 혼자 사는 청년인데, 행복주택 신청할 수 있어?"
ZIPFIT: "네, 도와드리겠습니다. 몇 가지 정보를 확인할게요..."
```

### 1.3.2 자격 진단 (Eligibility Check)

**기능**: 사용자 조건을 분석하여 신청 가능 여부 자동 판단

**주요 특징**:
- 공고문에서 자격 요건 자동 추출
- 사용자 정보와 자격 요건 매칭
- 충족/불충족 조건 명확히 표시

**예시**:
```
자격 조건 분석 (출처: LH 행복주택 공고문 p.3)
- 연령: 만 19~39세 (충족)
- 무주택 요건 (충족)
- 소득: 도시근로자 월평균소득 120% 이하 (충족)
```

### 1.3.3 공고 비교/추천 (Announcement Comparison & Recommendation)

**기능**: 사용자 조건에 맞는 공고를 추천하고 비교 분석

**주요 특징**:
- 여러 공고의 보증금, 월세, 위치, 입주 시기 등 비교
- 장단점 분석 및 추천 근거 제시
- 경쟁률, 우선순위 정보 제공

**예시**:
```
구분        | 성남 판교   | 수원 광교   | 수원 호매실
보증금      | 2,000만원   | 1,800만원   | 1,500만원
월세        | 30만원      | 28만원      | 25만원
경쟁률(예상)| 높음        | 중간        | 낮음
```

### 1.3.4 신청 안내 (Application Guidance)

**기능**: 단계별 신청 절차 상세 안내

**주요 특징**:
- 필요 서류 목록 제공
- 온라인 신청 방법 안내
- 일정 (접수, 발표, 계약) 정리
- 외부 사이트 (LH 청약센터 등) 직접 연결

**예시**:
```
1단계: 서류 준비
- 주민등록등본 (최근 3개월 이내)
- 소득증빙서류
- 무주택확인서

2단계: 온라인 접수
- LH 청약센터 접속
...
```

### 1.3.5 대출 정보 제공 (Loan Information)

**기능**: 정부 지원 대출 상품 정보 제공 및 연결

**주요 특징**:
- 주택도시기금 대출 상품 안내
- 대출 한도, 금리, 상환 방식 비교
- 대출 신청 절차 안내
- 금융기관 연결

**예시**:
```
신혼부부 전세자금 (주택도시기금)
- 한도: 최대 3억원
- 금리: 연 1.2~2.7%
- 상환: 10년 거치 후 10년 분할상환
```

### 1.3.6 용어 설명 (Glossary)

**기능**: 어려운 부동산 용어를 쉽게 설명

**주요 특징**:
- 1000개 부동산 용어 데이터베이스
- 쉬운 언어로 풀어쓴 설명
- 예시 및 관련 용어 제공

**예시**:
```
"묵시적갱신"이란?
계약 기간이 끝났는데 아무 말 없이 계속 살면
자동으로 계약이 연장되는 것입니다.
```

## 1.4 서비스 흐름 (User Journey)

```
[1. 진입]
사용자 → ZIPFIT 접속

[2. 정보 입력]
ZIPFIT → 사용자 정보 수집 (나이, 소득, 가족 구성 등)

[3. AI 상담]
사용자 ↔ ZIPFIT: 대화를 통한 니즈 파악

[4. 자격 진단]
ZIPFIT → 공고문 분석 → 자격 요건 자동 확인

[5. 공고 추천]
ZIPFIT → 맞춤형 공고 목록 제시

[6. 공고 비교]
사용자 → 관심 공고 선택
ZIPFIT → 상세 정보 및 장단점 분석 제공

[7. 신청 안내]
ZIPFIT → 신청 절차, 필요 서류, 일정 안내

[8. 대출 정보]
ZIPFIT → 대출 상품 정보 제공

[9. 외부 연결]
ZIPFIT → LH 청약센터, 주택도시기금 등 연결

[10. 계약 지원] (향후 확장)
ZIPFIT → 전자계약 시스템 연동
```

## 1.5 차별화 포인트

### 1.5.1 기존 서비스 vs ZIPFIT

**기존 공공주택 플랫폼의 한계**:
1. 공고를 단순 나열만 할 뿐, 내 조건에 맞는지 판단 불가
2. 각 기관(LH, SH, GH)별로 사이트가 분산되어 통합 검색 어려움
3. 전문 용어가 많아 이해하기 어려움
4. 신청부터 대출까지 여러 사이트를 오가야 하는 번거로움

**ZIPFIT의 차별점**:
1. **AI 기반 맞춤형 추천**: 사용자 조건을 분석하여 신청 가능한 공고만 필터링
2. **통합 플랫폼**: LH, SH, GH 공고를 한 곳에서 통합 검색
3. **쉬운 언어로 설명**: 복잡한 공고문을 AI가 쉽게 풀어서 설명
4. **원스톱 서비스**: 공고 검색 → 신청 → 대출까지 하나의 플랫폼에서 완결
5. **RAG 기반 정확성**: 실제 공고문 PDF를 분석하여 정확한 정보 제공 (할루시네이션 방지)

### 1.5.2 기술적 강점

**LLM + RAG 결합**:
- LLM의 자연어 이해 능력 + RAG의 정확한 문서 참조
- 할루시네이션 방지: 실제 공고문에 근거한 답변만 제공

**Vector DB 활용**:
- 공고문 전체를 벡터화하여 유사도 검색
- 빠른 검색 속도와 정확한 매칭

**통합 데이터 규격**:
- LH, SH, GH의 서로 다른 공고 형식을 통일된 규격으로 정리
- 기관 간 공고 비교 가능

## 1.6 프로젝트 범위

### Phase 1: 프로토타입 (2025.11.21 ~ 11.23)
**범위**:
- 데이터: 서울 LH 임대 공고만 (2024년 - 현재까지 올라온 공고만)
- 기능: 챗봇 + 공고 목록 + 상세보기

**목표**:
- 핵심 기능 시연
- 공모전 제출용 데모

### Phase 2: 확장 (예정)
**범위**:
- 데이터: 전체 지역, SH/GH 추가, 분양 추가
- 기능: 공고 비교, 자격 진단, 대출 정보 연동

**목표**:
- 실사용 가능한 서비스 구축

### Phase 3: 고도화 (예정)
**범위**:
- 데이터: 실시간 업데이트, 커뮤니티 리뷰
- 기능: 전자계약 연동, 알림 기능, 개인화 추천

**목표**:
- 상용 서비스 출시
- 연결형/융합형 확장 (지도, 생활 인프라, 학군 정보 등)

## 1.7 기대 사용자

### 주요 타겟
1. **청년 (만 19~39세)**
   - 행복주택, 청년 매입임대 등 관심
   - 주거 비용 부담 완화 필요

2. **신혼부부 (혼인 7년 이내)**
   - 신혼희망타운, 신혼부부 우선 공급 관심
   - 출산 준비 및 안정적 주거 필요

3. **저소득층/사회 취약계층**
   - 영구임대, 국민임대 등 관심
   - 주거 안정 및 복지 지원 필요

### 부가 타겟
4. **고령층**
   - 고령자 복지주택 관심
   - 디지털 격차로 인한 정보 접근 어려움

5. **다문화 가정**
   - 다양한 언어 지원 (향후 확장)
   - 주거 정보 이해도 향상 필요
