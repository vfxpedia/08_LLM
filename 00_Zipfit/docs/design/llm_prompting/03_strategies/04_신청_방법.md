# 신청 방법 안내 전략

## 개요

신청 방법 안내는 사용자가 공고 선택 후 실제 신청까지 원활하게 진행할 수 있도록 단계별로 안내하는 기능입니다.

---

## 1. 신청 절차 개요

### 1.1 일반적인 신청 절차 (LH 행복주택 기준)

```
[1단계] 서류 준비
- 주민등록등본
- 소득 증빙 서류
- 무주택 서약서
- 기타 자격 증빙 서류

[2단계] 온라인 신청
- LH 청약센터 접속 (apply.lh.or.kr)
- 회원가입 및 로그인
- 해당 공고 검색
- 신청서 작성 및 제출

[3단계] 당첨자 발표
- 발표일 확인
- 당첨 여부 조회
- 예비 순번 확인

[4단계] 계약 체결
- 계약 기간 확인
- 필요 서류 준비
- 현장 방문 계약

[5단계] 입주
- 입주 일정 안내
- 입주 점검
- 관리비 납부
```

## 2. 신청 방법 질문 유형

### 2.1 전체 절차 질문

**질문 예시**
- "신청은 어떻게 해?"
- "신청 방법 알려줘"
- "어디서 신청해?"

**응답 전략**
```
지피: LH 행복주택 신청 절차를 안내해드리겠습니다.

[1단계] 서류 준비
- 주민등록등본
- 소득 증빙 서류
- 무주택 서약서

[2단계] 온라인 신청
- LH 청약센터(apply.lh.or.kr) 접속
- 회원가입 및 로그인
- 해당 공고 선택 후 신청

[3단계] 결과 확인
- 당첨자 발표: 2024.12.10
- 계약 체결: 2024.12.15~12.20

(출처: LH_마포공덕_행복주택.pdf p.8-10)

```

## 5. 자주 묻는 질문 (FAQ)

### 5.1 신청 관련 FAQ

**Q1: 서류를 다 준비 못했는데 신청할 수 있어?**
```
지피: 네, 신청 가능합니다.

온라인 신청 시에는 서류 제출이 필요 없습니다.
서류는 당첨 후 계약 체결 시에 제출하면 됩니다.

[신청 절차]
1. 온라인 신청 (서류 불필요)
2. 당첨자 발표
3. 계약 시 서류 제출

단, 계약 시 서류가 없으면 계약이 불가능하므로
미리 준비하시는 것을 권장합니다.

(출처: LH_마포공덕_행복주택.pdf p.8-9)
```

**Q2: 중복 신청 가능해?**
```
지피: 동일한 공고에는 한 번만 신청 가능합니다.

단, 다른 공고에는 동시 신청 가능합니다.

[예시]
- 마포 공덕 행복주택: 신청 O
- 성동 행당 행복주택: 신청 O (동시 가능)
- 마포 공덕 (청년 + 신혼부부): 신청 X (중복 불가)

참고: 귀하는 청년과 신혼부부 두 자격에 해당하므로
더 유리한 청년 자격으로 신청하시길 권장합니다.

(출처: LH_마포공덕_행복주택.pdf p.7)
```

**Q3: 모바일로도 신청 가능해?**
```
지피: 네, 모바일로도 신청 가능합니다.

[모바일 신청]
1. 스마트폰 브라우저에서 apply.lh.or.kr 접속
2. 로그인 후 신청 (PC와 동일)

[LH 청약센터 앱]
- 앱 다운로드 가능 (구글 플레이스토어, 앱스토어)
- 앱에서도 신청 가능

(출처: LH 청약센터 이용 안내)

모바일 신청 방법을 상세히 안내해드릴까요?
```

## 6. 신청 체크리스트

### 6.1 신청 전 체크리스트

```
지피: 신청 전 체크리스트입니다.

[ ] 자격 조건 확인
    - 연령, 소득, 무주택 요건 충족

[ ] LH 청약센터 회원가입
    - apply.lh.or.kr

[ ] 필요 서류 확인
    - 주민등록등본, 소득 증빙, 무주택 서약서

[ ] 신청 기간 확인
    - 2024.11.18 ~ 2024.11.25 18:00

[ ] 자격 유형 선택
    - 청년 / 신혼부부 / 기타 (유리한 것 선택)

모두 준비되셨으면 신청 방법을 안내해드리겠습니다.
```

### 6.2 신청 후 체크리스트

```
지피: 신청을 완료하셨습니다.

다음 일정을 확인하세요:

[ ] 당첨자 발표: 2024.12.10 14:00
    - LH 청약센터에서 조회

[ ] 계약 체결: 2024.12.15 ~ 12.20
    - 마포구청 별관 2층

[ ] 서류 준비
    - 주민등록등본, 소득 증빙 원본, 무주택 서약서

[ ] 계약금 준비
    - 보증금: 1,200만원

(출처: LH_마포공덕_행복주택.pdf p.10)

알림 설정을 하시겠습니까?
```

## 7. 신청 안내 체크리스트

모든 신청 안내 시:

**정보 정확성**
- [ ] 신청 기간이 정확한가?
- [ ] 신청 사이트 URL이 맞는가?
- [ ] 필요 서류 목록이 정확한가?
- [ ] 계약 일정이 정확한가?

**단계별 안내**
- [ ] 단계별로 명확히 구분했는가?
- [ ] 각 단계의 필요 사항을 설명했는가?
- [ ] 다음 단계를 안내했는가?

**출처 명시**
- [ ] 모든 정보에 출처가 있는가?
- [ ] 페이지 번호가 정확한가?

**사용자 지원**
- [ ] 추가 질문을 유도했는가?
- [ ] 마감 임박 시 알림을 제공했는가?
- [ ] 체크리스트를 제공했는가?

```
사용자: 필요한 서류 자세히 알려줘
```
