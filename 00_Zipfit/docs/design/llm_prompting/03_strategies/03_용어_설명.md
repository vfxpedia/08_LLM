# 용어 설명 전략

## 개요

ZIPFIT은 1000개 부동산 용어 데이터베이스(Glossary_1000)를 활용하여 사용자가 공고문의 전문 용어를 쉽게 이해할 수 있도록 돕습니다. 단, Glossary에 없는 용어도 LLM의 일반 지식으로 보충 설명 가능합니다.

---

## 1. 용어 사전 구조

### 1.1 Glossary_1000 데이터 구조

```json
{
  "term": "묵시적갱신",
  "category": "계약_갱신",
  "definition_easy": "계약 기간이 끝났는데 아무 말 없이 계속 살면 자동으로 계약이 연장되는 것.",
  "definition_full": "임대차 기간 만료 전 당사자 어느 쪽도 갱신 거절 의사를 표시하지 않으면, 종전 계약과 동일한 조건으로 다시 계약한 것으로 봅니다.",
  "example": "2년 계약 만료 후 아무 통보 없으면 자동 2년 연장.",
  "related_terms": ["계약갱신청구권", "기간만료통지"]
}
```

### 1.2 데이터 활용

**검색**
- Vector DB에 임베딩 저장
- 용어 질문 시 유사도 검색
- 정확한 용어명 매칭 우선

**출력**
- 쉬운 정의 우선 제공
- 필요 시 전문 정의 추가
- 예시 제공 (있는 경우)
- 관련 용어 연결

## 2. 용어 설명 시나리오

### 2.1 직접 용어 질문

**시나리오**
```
사용자: 묵시적갱신이 뭐야?
```

**응답 구조**
```
[간단 정의]
묵시적갱신은 [쉬운 설명]입니다.

[상세 설명] - 필요시
[전문 설명]

[예시] - 있는 경우
[구체적 예시]

[출처]
(출처: Glossary_1000 - [카테고리])

[관련 질문]
관련 질문이 더 있으신가요?
```

**예시**
```
사용자: 묵시적갱신이 뭐야?

지피: 묵시적갱신은 계약 기간이 끝났는데 아무 말 없이 계속 거주하면
자동으로 계약이 연장되는 제도입니다.

[상세]
임대차 기간 만료 전 당사자 어느 쪽도 갱신 거절 의사를 표시하지 않으면,
종전 계약과 동일한 조건으로 다시 계약한 것으로 봅니다.

[예시]
2년 계약 만료 후 집주인이나 임차인이 아무 통보를 하지 않으면
자동으로 2년 연장됩니다.

(출처: Glossary_1000 - 계약_갱신 카테고리)

관련 용어: 계약갱신청구권, 기간만료통지
더 알고 싶은 용어가 있으신가요?
```

### 2.2 공고 맥락 내 용어 질문

**시나리오**
```
사용자: 이 공고에서 "전용면적"이 뭐야?
```

**응답 구조**
```
[용어 정의]

[이 공고 적용]
이 공고의 [해당 정보]

[출처]
- 용어: Glossary_1000
- 공고: 문서명 p.페이지
```

**예시**
```
사용자: 이 공고에서 "전용면적"이 뭐야?

지피: 전용면적은 실제로 사용할 수 있는 순수 거주 공간입니다.
벽체 중심선 기준으로 측정하며, 공용 면적(복도, 계단 등)은 제외됩니다.

[이 공고 적용]
마포구 공덕동 행복주택 전용면적: 26㎡
(약 8평, 방 1개 + 주방 + 화장실 규모)

(출처: 용어 - Glossary_1000 / 면적 - LH_마포공덕_행복주택.pdf p.5)

면적 관련 다른 질문이 있으신가요?
```

### 2.3 여러 용어 연속 질문

**시나리오**
```
사용자: 전용면적이랑 공급면적 차이가 뭐야?
```

**응답 구조**
```
[용어 1 정의]

[용어 2 정의]

[차이점 설명]

[출처]
```

**예시**
```
사용자: 전용면적이랑 공급면적 차이가 뭐야?

지피: 두 용어의 차이를 설명드리겠습니다.

[전용면적]
실제로 사용 가능한 순수 거주 공간
(벽체 중심선 기준, 공용 공간 제외)

[공급면적]
전용면적 + 주거 공용면적 (복도, 계단, 엘리베이터 등)
실제 분양/임대 계약 시 기준이 되는 면적

[차이]
공급면적이 전용면적보다 넓습니다.
예: 전용 26㎡ → 공급 약 30~33㎡ (공용 포함)

(출처: Glossary_1000 - 면적_관련 카테고리)

이 공고의 면적 정보를 확인해드릴까요?
```

## 3. Glossary 없는 용어 처리

### 3.1 원칙

**Glossary 우선**
- Glossary에 있는 용어는 반드시 DB 기반 설명
- 일반 지식으로 대체 금지

**Glossary 없을 때**
- LLM 일반 지식으로 보충 설명 가능
- 단, 출처를 "일반적인 부동산 용어"로 명시
- 공고문에 설명이 있으면 공고문 우선

### 3.2 Glossary 없는 용어 예시

```
사용자: "특별공급"이 뭐야?

[Glossary 검색]
→ 용어 없음

[공고문 검색]
→ Vector DB에서 "특별공급" 관련 청크 검색

[응답 - 공고문 기반]
지피: 특별공급은 일반 공급과 별도로, 특정 조건을 충족한 사람에게
우선 공급하는 제도입니다.

이 공고의 특별공급 대상:
- 국가유공자
- 장애인
- 다자녀 가구

(출처: LH_마포공덕_행복주택.pdf p.4)

자격 조건을 확인해드릴까요?
```

```
사용자: "주택청약종합저축"이 뭐야?

[Glossary 검색]
→ 용어 없음

[공고문 검색]
→ 관련 내용 없음

[응답 - 일반 지식]
지피: 주택청약종합저축은 공공주택 분양/임대를 신청하기 위해
가입하는 저축 상품입니다.

일반적으로:
- 은행에서 가입
- 매월 일정 금액 납입
- 청약 자격 획득

단, 이 공고(행복주택)는 청약통장이 필요하지 않습니다.
(출처: 일반적인 부동산 용어)

이 공고의 신청 자격을 확인해드릴까요?
```

## 4. 용어 설명 출력 형식

### 4.1 기본 형식

```
[용어명]: [간단 정의]

[상세 설명] - 필요시

[예시] - 있는 경우

(출처: Glossary_1000 - [카테고리])

[관련 용어] - 있는 경우
```

### 4.2 짧은 설명 (간단한 용어)

```
사용자: 보증금이 뭐야?

지피: 보증금은 임대차 계약 시 임차인이 집주인에게 맡기는 돈입니다.
계약 종료 시 반환받습니다.

(출처: Glossary_1000 - 임대차_기본)
```

### 4.3 긴 설명 (복잡한 용어)

```
사용자: 계약갱신청구권이 뭐야?

지피: 계약갱신청구권은 임차인이 계약 기간이 끝났을 때
한 번 더 계약을 연장할 수 있도록 요청할 수 있는 권리입니다.

[상세]
임대차 기간이 2년 이상이면, 임차인은 계약 만료 6개월 전부터
2개월 전까지 집주인에게 갱신을 요청할 수 있습니다.
집주인은 정당한 사유 없이 거절할 수 없습니다.

[조건]
- 최초 계약 후 1회만 가능 (총 4년 거주 가능)
- 임대료는 5% 이내 인상 가능

[예시]
2023년 1월 계약 (2년) → 2024년 11월 갱신 요청 가능
→ 2025년 1월부터 2년 연장 (총 4년 거주)

(출처: Glossary_1000 - 계약_갱신 카테고리)

행복주택에도 계약갱신청구권이 적용됩니다.
더 궁금한 점이 있으신가요?
```

## 5. 관련 용어 연결

### 5.1 관련 용어 제시

**Glossary에 관련 용어 있을 때**
```
사용자: 묵시적갱신이 뭐야?

지피: 묵시적갱신은 계약 기간이 끝났는데 아무 말 없이 계속 거주하면
자동으로 계약이 연장되는 제도입니다.

[상세 설명...]

(출처: Glossary_1000 - 계약_갱신 카테고리)

관련 용어:
- 계약갱신청구권: 임차인이 계약 연장을 요청할 수 있는 권리
- 기간만료통지: 계약 만료 전 상대방에게 통지하는 것

더 알고 싶은 용어가 있으신가요?
```

### 5.2 연속 질문 유도

```
사용자: 계약갱신청구권 설명해줘

지피: [계약갱신청구권 설명...]

(출처: Glossary_1000)

관련 용어:
- 묵시적갱신
- 임대료 인상 제한

이 중 궁금한 용어가 있으신가요?
```

## 6. 용어 + 공고 정보 결합

### 6.1 용어 설명 후 공고 적용

```
사용자: 도시근로자 월평균소득이 뭐야?

지피: 도시근로자 월평균소득은 통계청에서 매년 발표하는
전국 도시 지역 근로자 가구의 평균 소득입니다.

공공주택 신청 시 소득 기준으로 사용됩니다.

[이 공고 적용]
마포구 공덕동 행복주택 소득 기준:
- 1인 가구: 도시근로자 월평균소득 120% 이하
- 2024년 기준: 약 360만원

귀하의 소득(250만원)은 기준을 충족합니다.

(출처: 용어 - Glossary_1000 / 기준 - LH_마포공덕_행복주택.pdf p.3)

자격 조건을 상세히 확인해드릴까요?
```

### 6.2 공고 용어 문의

```
사용자: 공고에 "무주택세대구성원"이라고 나오는데 뭐야?

지피: 무주택세대구성원은 본인과 배우자, 그리고 같이 사는 가족
전원이 집을 소유하지 않은 사람을 의미합니다.

[포함 대상]
- 세대주
- 배우자
- 직계존속 (부모, 조부모)
- 직계비속 (자녀, 손자녀)

[이 공고 적용]
마포구 공덕동 행복주택은 무주택세대구성원만 신청 가능합니다.

(출처: 용어 - Glossary_1000 / 공고 - LH_마포공덕_행복주택.pdf p.3)

무주택 요건을 충족하시나요?
```

## 7. 용어 설명 체크리스트

모든 용어 설명 시:

**검색 단계**
- [ ] Glossary_1000에 있는 용어인가?
- [ ] 공고문에 설명이 있는가?
- [ ] 일반 지식으로 보충해야 하는가?

**설명 단계**
- [ ] 쉬운 정의를 먼저 제공했는가?
- [ ] 필요시 상세 설명을 추가했는가?
- [ ] 예시를 제공했는가? (있는 경우)

**출처 단계**
- [ ] Glossary 용어는 출처를 명시했는가?
- [ ] 일반 지식은 "일반적인 부동산 용어"로 명시했는가?

**연결 단계**
- [ ] 관련 용어를 제시했는가?
- [ ] 공고 정보와 연결했는가? (해당 시)
- [ ] 추가 질문을 유도했는가?

## 8. 예시

### 예시 1: Glossary 용어 - 기본

```
사용자: 전세가 뭐야?

지피: 전세는 집을 빌릴 때 보증금만 내고 월세는 내지 않는 임대 방식입니다.

[상세]
계약 기간 동안 목돈(전세금)을 집주인에게 맡기고,
계약 종료 시 전액을 돌려받습니다.

[예시]
2억원 전세: 2억원 보증금만 내고, 월세 0원
2년 후 계약 종료 시 2억원 반환

(출처: Glossary_1000 - 임대차_유형)

참고: 행복주택은 전세가 아닌 보증금+월세 방식입니다.
```

### 예시 2: Glossary 용어 - 공고 연결

```
사용자: 우선공급이 뭐야?

지피: 우선공급은 특정 조건을 충족한 사람에게 일반 공급보다
먼저 공급하는 제도입니다.

[이 공고 적용]
마포구 공덕동 행복주택 우선공급 대상:
- 해당 지역 거주자
- 국가유공자
- 장애인

(출처: 용어 - Glossary_1000 / 공고 - LH_마포공덕_행복주택.pdf p.4)

귀하가 우선공급 대상인지 확인해드릴까요?
```

### 예시 3: Glossary 없는 용어 - 일반 지식

```
사용자: 입주금이 뭐야?

[Glossary 검색] → 없음
[공고문 검색] → 없음

지피: 입주금은 분양주택 계약 시 납부하는 금액입니다.

일반적으로:
- 계약금 (10%)
- 중도금 (60%)
- 잔금 (30%)
순서로 납부합니다.

(출처: 일반적인 부동산 용어)

참고: 이 공고(행복주택)는 임대이므로 입주금이 아닌
보증금과 월세만 납부합니다.

보증금/월세 정보를 확인해드릴까요?
```

### 예시 4: 복수 용어 비교

```
사용자: 전용면적, 공급면적, 계약면적 차이가 뭐야?

지피: 세 가지 면적 용어의 차이를 설명드리겠습니다.

[전용면적]
실제 사용 가능한 순수 거주 공간
(벽체 중심선 기준)

[공급면적]
전용면적 + 주거 공용면적 (복도, 계단 등)

[계약면적]
공급면적 + 기타 공용면적 (주차장, 관리실 등)

[크기 순서]
전용 < 공급 < 계약

[예시]
전용 26㎡ → 공급 30㎡ → 계약 33㎡

(출처: Glossary_1000 - 면적_관련)

이 공고의 면적 정보를 확인해드릴까요?
```
