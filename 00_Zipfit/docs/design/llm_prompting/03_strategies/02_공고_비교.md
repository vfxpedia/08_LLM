# 공고 비교 전략

## 개요

공고 비교는 사용자가 여러 선택지 중 최적의 주거를 선택할 수 있도록 돕는 핵심 기능입니다. **최대 3개 공고**까지만 비교하여 정보 과부하를 방지합니다.

---

## 1. 비교 원칙

### 1.1 최대 3개 제한

**이유**
- 정보 과부하 방지
- 사용자 의사결정 혼란 방지
- 명확한 비교 가능

**3개 초과 시 대응**
```
사용자: 서울 공고 전부 비교해줘

지피: 현재 서울 지역 모집 중인 공고가 12건입니다.

한 번에 비교하기에는 정보가 많아 혼란스러울 수 있습니다.
다음 조건 중 중요한 것을 선택해주시면 맞춤 비교를 도와드리겠습니다:

1. 지역 (강남권/강북권/도심권)
2. 월세 범위 (20만원 이하/20~30만원)
3. 입주 시기 (3개월 내/6개월 내)

어느 조건을 우선하시나요?
```

### 1.2 객관적 비교

**원칙**
- 주관적 추천 금지
- 사실 기반 비교만
- 장단점 균형 있게 제시

**잘못된 예**
```
마포가 훨씬 좋아요! 성동은 별로예요.
(X - 주관적 판단)
```

**올바른 예**
```
마포 vs 성동 비교:

마포: 교통 우수, 월세 높음
성동: 월세 저렴, 역에서 도보 10분

교통과 월세 중 어느 쪽을 우선하시나요?
(O - 객관적 비교, 선택권 존중)
```

## 2. 비교 기준

### 2.1 핵심 비교 항목

**필수 비교 항목**
1. 비용 (보증금, 월세)
2. 위치 (지역, 교통 접근성)
3. 입주 시기
4. 면적 및 구조

**부가 비교 항목 (문서 명시 시)**
5. 신청 마감일
6. 공급 세대수
7. 자격 조건
8. 가점 배점

### 2.2 사용자 우선순위

**우선순위 파악**
```
[1순위] 비용
- "월세 저렴한 곳"
- "보증금 낮은 곳"

[2순위] 위치
- "강남권 공고"
- "교통 편한 곳"

[3순위] 입주 시기
- "빨리 들어갈 수 있는 곳"
- "3개월 내 입주"
```

**우선순위 확인 질문**
```
지피: 비교 시 다음 중 가장 중요한 조건은 무엇인가요?

1. 비용 (보증금/월세)
2. 위치 (지역/교통)
3. 입주 시기

어느 것을 우선하시나요?
```

## 3. 비교 출력 형식

### 3.1 기본 비교 형식

**구조**
```
[비교 개요]

[공고 1 정보]
- 항목 1
- 항목 2
- 항목 3

[공고 2 정보]
- 항목 1
- 항목 2
- 항목 3

[출처]
[선택 유도 질문]
```

**예시**
```
지피: 두 공고를 비교해드리겠습니다.

마포구 공덕동 행복주택:
- 보증금/월세: 1,200만원 / 22만원
- 교통: 지하철 5,6호선 도보 5분
- 입주: 2025년 3월
- 면적: 26㎡

성동구 행당동 행복주택:
- 보증금/월세: 1,000만원 / 19만원
- 교통: 지하철 2,5호선 도보 10분
- 입주: 2025년 5월
- 면적: 24㎡

(출처: 각 공고문 p.5)

교통 편리성과 월세 중 어느 쪽을 우선하시나요?
```

### 3.2 표 형식 비교

**활용**
- 3개 공고 비교 시
- 수치 정보 비교 시
- 한눈에 비교 필요 시

**예시**
```
지피: 세 공고를 비교해드리겠습니다.

| 항목 | 마포 공덕 | 성동 행당 | 강북 수유 |
|------|-----------|-----------|-----------|
| 보증금 | 1,200만원 | 1,000만원 | 800만원 |
| 월세 | 22만원 | 19만원 | 18만원 |
| 면적 | 26㎡ | 24㎡ | 22㎡ |
| 입주 | 2025.03 | 2025.05 | 2025.06 |
| 교통 | 도보 5분 | 도보 10분 | 도보 15분 |

(출처: 각 공고문)

어느 조건을 우선하시나요?
```

### 3.3 장단점 비교

**활용**
- 복잡한 선택 시
- 트레이드오프 명확화
- 의사결정 지원

**예시**
```
지피: 마포 vs 성동 장단점 비교:

마포 공덕:
✓ 장점: 교통 편리 (역세권), 빠른 입주
✗ 단점: 월세 높음 (22만원), 경쟁률 높음

성동 행당:
✓ 장점: 월세 저렴 (19만원), 경쟁률 낮음
✗ 단점: 역에서 도보 10분, 입주 늦음

(출처: 각 공고문)

교통과 월세 중 무엇을 우선하시나요?
```

## 4. 비교 시나리오

### 4.1 2개 공고 비교

**일반적 비교**
```
사용자: 마포랑 성동 비교해줘

[RDB 검색]
→ 마포 공고: LH_lease_1
→ 성동 공고: LH_lease_2

[Vector DB 검색]
→ 각 공고의 보증금/월세, 교통, 입주일 등

지피: 두 공고를 비교해드리겠습니다.

마포 공덕:
- 보증금/월세: 1,200만원 / 22만원
- 교통: 지하철 5,6호선 도보 5분
- 입주: 2025년 3월

성동 행당:
- 보증금/월세: 1,000만원 / 19만원
- 교통: 지하철 2,5호선 도보 10분
- 입주: 2025년 5월

(출처: 각 공고문 p.5)

어느 조건을 우선하시나요?
```

### 4.2 3개 공고 비교

**최대 허용 비교**
```
사용자: 마포, 성동, 강북 비교해줘

[RDB 검색]
→ 3개 공고 필터링

[Vector DB 검색]
→ 각 공고 상세 정보

지피: 세 공고를 비교해드리겠습니다.

| 항목 | 마포 | 성동 | 강북 |
|------|------|------|------|
| 보증금 | 1,200 | 1,000 | 800 |
| 월세 | 22 | 19 | 18 |
| 교통 | ★★★ | ★★ | ★ |
| 입주 | 2025.03 | 2025.05 | 2025.06 |

(출처: 각 공고문)

우선 조건을 알려주시면 더 구체적인 비교를 도와드리겠습니다.
```

### 4.3 조건별 필터 후 비교

**4개 이상 시**
```
사용자: 서울에서 월세 20만원 이하 공고 비교해줘

[RDB 검색]
→ 서울 + 월세 ≤ 200,000원
→ 결과: 5개 공고

지피: 서울 지역 월세 20만원 이하 공고 5건이 있습니다.

추가 조건으로 좁혀드릴까요?

1. 지역 (강남권/강북권/도심권)
2. 입주 시기 (3개월 내/6개월 내)
3. 면적 (20㎡ 이하/20~30㎡/30㎡ 이상)

어느 조건을 우선하시나요?

사용자: 강남권

지피: 강남권 월세 20만원 이하 공고 3건 비교:

| 공고 | 보증금 | 월세 | 입주 |
|------|--------|------|------|
| 강남 A | 1,100 | 20 | 2025.04 |
| 송파 B | 1,000 | 19 | 2025.05 |
| 서초 C | 1,200 | 18 | 2025.06 |

(출처: 각 공고문)
```

## 5. 우선순위별 비교 전략

### 5.1 비용 우선

**사용자 의도**
- "월세 저렴한 곳"
- "보증금 낮은 곳"
- "비용 비교해줘"

**비교 전략**
```
지피: 월세 기준 비교:

1. 강북 수유: 18만원 (보증금 800만원)
2. 성동 행당: 19만원 (보증금 1,000만원)
3. 마포 공덕: 22만원 (보증금 1,200만원)

강북이 가장 저렴하지만, 교통 접근성이 낮습니다 (역에서 도보 15분).
성동은 중간 가격대에 교통도 적당합니다 (역에서 도보 10분).

(출처: 각 공고문 p.5)

교통과 비용 중 어느 쪽을 우선하시나요?
```

### 5.2 위치 우선

**사용자 의도**
- "역세권 공고"
- "교통 편한 곳"
- "강남권 공고"

**비교 전략**
```
지피: 교통 접근성 기준 비교:

1. 마포 공덕: 지하철 5,6호선 도보 5분 ★★★
   (월세 22만원)

2. 성동 행당: 지하철 2,5호선 도보 10분 ★★
   (월세 19만원)

3. 강북 수유: 지하철 4호선 도보 15분 ★
   (월세 18만원)

마포가 가장 편리하지만, 월세가 가장 높습니다.

(출처: 각 공고문 p.5)

교통 우선이신가요, 비용 우선이신가요?
```

### 5.3 입주 시기 우선

**사용자 의도**
- "빨리 들어갈 수 있는 곳"
- "3개월 내 입주"
- "입주 빠른 곳"

**비교 전략**
```
지피: 입주 시기 기준 비교:

1. 마포 공덕: 2025년 3월 (2개월 후)
   - 월세: 22만원
   - 교통: 도보 5분

2. 성동 행당: 2025년 5월 (4개월 후)
   - 월세: 19만원
   - 교통: 도보 10분

3. 강북 수유: 2025년 6월 (5개월 후)
   - 월세: 18만원
   - 교통: 도보 15분

마포가 가장 빠르지만, 월세가 높고 경쟁률도 높을 것으로 예상됩니다.

(출처: 각 공고문)

빠른 입주가 우선이신가요?
```

## 6. 특수 비교 시나리오

### 6.1 동일 공고 내 여러 유형

**30개 이상 세대 공고**
```
사용자: 이 공고에 어떤 옵션들이 있어?

지피: 마포구 공덕동 행복주택은 총 120세대로, 3가지 유형이 있습니다.

| 유형 | 세대수 | 면적 | 월세 | 대상 |
|------|--------|------|------|------|
| A타입 | 80 | 26㎡ | 22만원 | 청년 |
| B타입 | 30 | 36㎡ | 30만원 | 신혼부부 |
| C타입 | 10 | 46㎡ | 40만원 | 일반 |

(출처: LH_마포공덕_행복주택.pdf p.5)

희망하시는 면적이나 대상이 있으신가요?
```

### 6.2 자격별 비교

**여러 자격 해당 시**
```
사용자: 청년이랑 신혼부부 중 뭐가 나아?

지피: 귀하는 두 자격 모두 해당하십니다.

[청년 자격]
- 월세: 22만원 (26㎡)
- 가점: 30점
- 경쟁률: 높음

[신혼부부 자격]
- 월세: 30만원 (36㎡)
- 가점: 25점
- 경쟁률: 중간

청년 자격이:
- 월세 8만원 저렴
- 가점 5점 높음
- 하지만 면적 10㎡ 작음

(출처: LH_마포공덕_행복주택.pdf p.5, p.12)

월세와 면적 중 어느 것을 우선하시나요?
```

## 7. 비교 후 다음 단계

### 7.1 선택 유도

```
지피: 비교 결과를 보셨습니다.

다음 중 필요한 정보가 있으신가요?

1. 특정 공고 상세 정보
2. 자격 조건 확인
3. 신청 방법 안내
4. 다른 공고와 추가 비교

무엇을 도와드릴까요?
```

### 7.2 추가 정보 제공

```
사용자: 마포가 좋을 것 같아

지피: 마포구 공덕동 행복주택 상세 정보:

- 보증금/월세: 1,200만원 / 22만원
- 면적: 26㎡
- 입주: 2025년 3월
- 교통: 지하철 5,6호선 도보 5분
- 신청 마감: 2024.11.25 (3일 남음)

(출처: LH_마포공덕_행복주택.pdf p.5)

자격 조건을 확인해드릴까요?
```

## 8. 비교 체크리스트

모든 비교 시:

**비교 전**
- [ ] 비교 대상이 3개 이하인가?
- [ ] 3개 초과 시 조건 좁히기 안내했는가?
- [ ] 사용자 우선순위를 파악했는가?

**비교 중**
- [ ] 객관적 정보만 제시했는가?
- [ ] 주관적 추천을 하지 않았는가?
- [ ] 모든 공고에 동일한 항목을 비교했는가?
- [ ] 출처를 명시했는가?

**비교 후**
- [ ] 장단점을 균형 있게 제시했는가?
- [ ] 선택 유도 질문을 했는가?
- [ ] 다음 단계를 안내했는가?

## 9. 예시

### 예시 1: 2개 비교 - 비용 우선

```
사용자: 마포랑 성동 중 어디가 저렴해?

지피: 비용 비교:

성동 행당이 더 저렴합니다.

- 성동: 보증금 1,000만원 / 월세 19만원
- 마포: 보증금 1,200만원 / 월세 22만원

차이:
- 보증금 200만원 차이
- 월세 3만원 차이 (연간 36만원)

(출처: 각 공고문 p.5)

단, 마포는 교통 접근성이 우수합니다 (역세권).
교통과 비용 중 어느 쪽을 우선하시나요?
```

### 예시 2: 3개 비교 - 종합

```
사용자: 마포, 성동, 강북 비교해줘

지피: 세 공고를 비교해드리겠습니다.

| 항목 | 마포 공덕 | 성동 행당 | 강북 수유 |
|------|-----------|-----------|-----------|
| 보증금 | 1,200만원 | 1,000만원 | 800만원 |
| 월세 | 22만원 | 19만원 | 18만원 |
| 교통 | 도보 5분 | 도보 10분 | 도보 15분 |
| 입주 | 2025.03 | 2025.05 | 2025.06 |

종합:
- 마포: 교통 최우수, 빠른 입주, 비용 높음
- 성동: 중간 가격, 적당한 교통
- 강북: 가장 저렴, 교통 낮음, 늦은 입주

(출처: 각 공고문)

비용, 교통, 입주 시기 중 가장 중요한 조건은 무엇인가요?
```
