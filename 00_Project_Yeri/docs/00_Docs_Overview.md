# 00_Docs_Overview.md

이 파일은 docs에 대한 설명을 담고 있다.

우리는 아래와 같은 규칙을 따른다.

`##_문서주요내용.md` 와 같은 형식으로 문서 파일명을 규약한다.<br>
최신 문서의 내용은 위와 동일하다.

이전 문서의 내용은 ##_문서주요내용.v##.md 와 같은 형식으로 문서 파일명을 규약한다.<br>
이를 삭제하지 않은 이유는 해당 문서에서도 프로젝트를 진행하는데 있어서 필요한 부분이 있을 것이라고 판단되기 때문이다.<br>
<br>
하지만 실제 개발에서 중점적인 것은 최신 문서를 기반으로 진행한다.
