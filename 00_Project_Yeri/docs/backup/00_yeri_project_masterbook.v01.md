# 예리는 못 말려: 프로젝트 마스터북.v01

## 1. 프로젝트 개요

이 프로젝트는 EEVE LLM을 중심 기술로 활용하며, 단순히 대사를 생성하는 수준이 아니라 예리의 감정 해석, 플레이어의 답변 분석, 센스 평가, 그리고 맞춤형 반응 생성을 담당하는 핵심 AI 로직을 포함한다.

### 1.1 프로젝트 개념
‘예리는 못 말려’는 AI 기반 인터랙티브 연애 시뮬레이션 게임으로, 플레이어가 AI 캐릭터 ‘예리’와 시각적·감정적 상호작용을 통해 몰입형 경험을 얻는 것을 목표로 한다. 단순히 “차이점을 맞추는 게임”이 아니라, 감정과 센스를 평가하는 **하이브리드 감정형 인터랙션 시스템**이다.

### 1.2 핵심 경험
- **핵심 문장:** “오빠, 나 뭐 달라진 거 없어?”
- **핵심 감정:** 관찰력 + 감정 센스 + 몰입 대화의 결합.
- **목표:** 시각 정보와 언어 감정 피드백을 융합한 대화형 연애 콘텐츠.

### 1.3 게임 키워드
- AI 연애 시뮬레이션
- 시각 감정 퀴즈
- TTS/STT 음성 인터랙션
- SNS 밈 공유형 콘텐츠

## 2. 문서 구성 및 개발 순서 (Top → Bottom)

EEVE를 핵심 기술 축으로 활용하는 기술적 설명 섹션을 강화한다. 이 LLM은 이미지 비교 결과, 사용자 답변, 감정 피드백을 통합 분석하여 예리의 반응을 생성하고 점수를 산출하는 역할을 수행한다. 이 구조를 명확히 기술하면 Runpod 환경에서의 LLM 활용 포인트가 구체적으로 드러날 것이다.

### 2.1 문서 구조 개요
이 프로젝트는 ‘마스터북’ 문서와 여러 개의 세부 문서로 구성된다.

| 문서명 | 주요 내용 |
|----------|-------------|
| **1. Game_Structure.md** | 전체 게임 흐름, 오프닝부터 클로징까지 구조 설계 |
| **2. Character_Design.md** | 예리 캐릭터 기획, 표정 변화, 감정 상태, 음성 톤 설계 |
| **3. UI_UX_Design.md** | 화면 구성, 상단 타임라인, 카드 뒤집기 연출 설계 |
| **4. Gameplay_Mechanics.md** | 점수 시스템, 타이머, 감정 반응, 피드백 로직 |
| **5. Technical_Implementation.md** | 기술 아키텍처, Ollama EEVE, 이미지 모듈, 음성 입출력 |
| **6. Production_Pipeline.md** | 개발 단계 로드맵, MVP → Beta → Release 순서 |
| **7. Marketing_SNS.md** | 결과 카드, SNS 확산 전략, 밈 챌린지 기획 |

---

### 2.2 문서 개발 순서

#### 1단계: Game_Structure.md
- 오프닝, 이미지 표시 방식, TTS 타이밍, 플레이 흐름, 점수 구조 설계.
- 타임라인(시간 감소) + 예리 표정 변화 포함.

#### 2단계: Character_Design.md
- 예리 캐릭터의 시각적/정서적 톤 확립.
- 시간 흐름에 따른 표정 변화(밝음 → 집중 → 울상) 로직 정의.

#### 3단계: UI_UX_Design.md
- 상단 타임라인 애니메이션 설계.
- 카드 뒤집기 효과, 사운드 연출(샤라랄라 효과) 세부화.

#### 4단계: Gameplay_Mechanics.md
- 관찰력, 감정 센스, 재치 센스, 속도 점수 계산 로직 확정.
- 시도 횟수(3회), 타임 제한(기본 10초) 구조 구체화.

#### 5단계: Technical_Implementation.md
- EEVE는 시각 분석 결과와 사용자 입력 텍스트를 기반으로 감정 스코어를 계산하고 자연스러운 대사를 생성한다.
- LLM은 이미지 비교 결과·사용자 답변·감정 피드백을 통합 분석하여 반응을 생성하고 점수를 산출한다.
- Ollama EEVE-Korean 연결.
- TTS(gTTS/ElevenLabs), STT(Whisper) 연동.
- 이미지 모듈 및 점수 DB 관리 구조.

#### 6단계: Production_Pipeline.md
- MVP → Prototype → Beta → Final 단계별 목표 및 일정 정의.
- 버전 관리 전략 및 QA 플로우 포함.

#### 7단계: Marketing_SNS.md
- SNS 밈 카드 템플릿 제작.
- 공유 문구, 해시태그, 챌린지 이벤트 시나리오.
