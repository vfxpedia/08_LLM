## ⚠️ 예상 문제 및 해결 방법

### 문제 1: `npm install` 실패
**원인**: Node.js 버전 문제
**해결**: Node.js 20 이상 설치 확인
```bash
node --version  # v20 이상이어야 함
```

### 문제 2: 페이지가 로딩되지 않음
**원인**: 포트 3000이 이미 사용 중
**해결**: 다른 포트로 실행
```bash
PORT=3001 npm run dev
```

### 문제 3: 이미지가 표시되지 않음
**원인**: 테스트 이미지가 없음
**해결**: 정상 동작 - 플레이스홀더(🖼️)가 표시되면 OK

### 문제 4: Tailwind CSS 스타일이 적용되지 않음
**원인**: `tailwind.config.ts`가 없음
**해결**: 이미 생성되어 있어야 함 - 개발 서버 재시작

### 문제 5: 음성 녹음이 작동하지 않음
**원인**: 브라우저가 Web Speech API를 지원하지 않음
**해결**: Chrome 또는 Edge 브라우저 사용

### 문제 6: TypeScript 에러
**원인**: 타입 정의 누락
**해결**: 다음 명령어로 에러 확인
```bash
npm run build
```
